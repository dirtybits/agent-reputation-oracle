# Coral Paper

A fully-light sibling of Coral Terminal. Same coral accent and monospace DNA, but the dark code blocks are swapped for soft paper cards with hairline rules — like reading a hand-set zine instead of a terminal. Clean, approachable, still technical.

## Color Palette

- **Coral**: `#F28A61` - Signature accent; titles, call-out chips, icon circles, emphasis
- **Ink**: `#1F2937` - Primary text and icons (softer than pure black)
- **Paper**: `#FAFAFA` - Primary background for all slides
- **Ash**: `#6B7280` - Muted body text, captions, metadata

### Supporting Tones

- **Ember**: `#FD522F` - Stronger coral for destructive / alert hits
- **Smoke**: `#D1D5DB` - Card borders and dividers (1pt hairlines, no drop shadows)
- **Cream**: `#FFF5EE` - Tinted card fill for emphasis regions (very subtle coral wash)
- **Amber**: `#F59E0B` - Secondary accent for status / tertiary callouts

## Typography

- **Headers**: Arial Black
- **Body Text**: Calibri
- **Monospace (signature)**: Inconsolata — used for eyebrows, code, technical labels, stat numerators. Even in the light variant, monospace carries the engineering identity.

## Signature Motifs

- **All-light surfaces**: every slide uses `#FAFAFA` background; no dark inversions
- **Hairline card system**: content grouped in `#D1D5DB` 1pt-bordered rectangles, no fills
- **Coral eyebrow chip** (all-caps mono, 10pt) above each title
- **Icon-in-coral-circle** stays the same motif as Terminal / Midnight
- **Code samples** rendered as Inconsolata on `#FFF5EE` cream blocks with coral left rule
- **Big stat callouts** in coral with Ink label below

## Best Used For

Product one-pagers, investor updates where "serious but readable" matters, documentation, research writeups, presentations printed to PDF, anything delivered to a mixed audience where dark slides would feel heavy.
