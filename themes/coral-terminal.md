# Coral Terminal

A hybrid light/dark "developer-native" theme that pairs cream content surfaces with near-black code blocks, unified by a coral accent. Think of it as a well-lit workshop with a dark-mode editor sitting on the bench — the workshop for your ideas, the editor for your proofs. Derived from the AgentVouch walkthrough deck.

## Color Palette

- **Coral**: `#F28A61` - Signature accent; titles, call-out chips, icon circles, key numbers
- **Ink**: `#030712` - Dark surfaces, code blocks, terminal panels, bold body
- **Paper**: `#F3F4F6` - Primary light background for content slides
- **Ash**: `#6B7280` - Muted body text, captions, metadata

### Supporting Tones

- **Ember**: `#FD522F` - Stronger coral for destructive / alert hits (slashing, warnings)
- **Smoke**: `#D1D5DB` - Hairline borders, dividers, low-emphasis rules
- **Slate**: `#1F2937` - Secondary dark (card fills inside dark regions)
- **Amber**: `#F59E0B` - Secondary accent for status / tertiary callouts

## Typography

- **Headers**: Arial Black
- **Body Text**: Calibri
- **Monospace (signature)**: Inconsolata — used for code, technical labels, "terminal" chips, and any content with a systems/crypto/engineering flavor. Mono typography is a load-bearing motif, not decoration.

## Signature Motifs

- **Sandwich structure**: coral title slide → light content slides → dark code/terminal slides → coral closing slide
- **Icon-in-coral-circle** next to section headers
- **Monospace labels** (all-caps, tracked) for section eyebrows like `THE PROBLEM` / `THE SOLUTION`
- **Dark code blocks** on light slides for emphasis (inverse: light code on dark slides)
- **Thin coral rule** top-left of content slides, not a full underline under the title
- **Big stat callouts** in coral (60-72pt) with small mono label below

## Best Used For

Crypto / web3 protocols, developer tools, AI agent infrastructure, security disclosures, technical walkthroughs, hackathon pitches, anything where engineering credibility matters more than polish.
