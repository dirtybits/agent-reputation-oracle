# Coral Midnight

A fully-dark sibling of Coral Terminal. The whole deck is the terminal — Ink surfaces everywhere, with coral used sparingly so it glows like the cursor on a dark shell. Premium, confident, and dense without feeling crowded.

## Color Palette

- **Coral**: `#F28A61` - Signature accent; titles, key numbers, active states (used sparingly — coral should glow)
- **Ink**: `#030712` - Primary background for every slide
- **Slate**: `#1F2937` - Card fills and elevated surfaces on the Ink background
- **Moon**: `#D1D5DB` - Body text and high-emphasis labels

### Supporting Tones

- **Ember**: `#FD522F` - Stronger coral for destructive / alert hits
- **Ash**: `#6B7280` - Muted body text, captions, metadata
- **Fog**: `#374151` - Hairline borders on Slate cards (1pt, subtle)
- **Amber**: `#F59E0B` - Secondary accent for status / tertiary callouts

## Typography

- **Headers**: Arial Black
- **Body Text**: Calibri
- **Monospace (signature)**: Inconsolata — runs the show. On a dark background, mono typography disappears into "this is a system we built," which is exactly the effect we want.

## Signature Motifs

- **Dark-throughout**: every slide is Ink; no light inversions (unlike Terminal's sandwich)
- **Slate cards** with `#374151` 1pt borders — the only structural element besides text
- **Coral eyebrow chip** (all-caps mono, 10pt) above each title
- **Icon-in-coral-circle** with Ink glyph inside (reverse of the light themes)
- **Code samples** appear on *lighter* Slate fills (the inversion of the Paper theme)
- **Oversized stat numbers** in coral with Moon label — coral is the only warm color, so the numbers pop
- **Minimal use of Ember / Amber** — accents should feel rare and intentional

## Best Used For

Keynote-style talks, security / infra product launches, late-night developer demos, pitch decks where you want the audience to lean forward, anything delivered in a darkened room or onto an OLED screen.
